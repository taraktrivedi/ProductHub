# Full Stack ProductHub Development Prompt

## Objective
Create a comprehensive full-stack Product Management tool called **ProductHub** that consolidates customer feedback, enables feature prioritization, and provides advanced roadmapping capabilities. The application should demonstrate real workflows and data flows using realistic dummy data.

## Technical Architecture Requirements

### Backend Stack
- **Framework**: Node.js with Express.js or Python with FastAPI
- **Database**: PostgreSQL with Redis for caching
- **Authentication**: JWT-based auth with role management
- **API Design**: RESTful APIs with comprehensive endpoints
- **File Storage**: Local storage with cloud-ready structure
- **AI Integration**: OpenAI/Claude API for idea generation

### Frontend Stack
- **Framework**: React.js with TypeScript
- **State Management**: Redux Toolkit or Zustand
- **UI Library**: Material-UI or Ant Design
- **Charts/Visualization**: D3.js, Recharts, or Chart.js
- **Drag & Drop**: React DnD for roadmap interface
- **Real-time**: WebSocket connections for live updates
